# Notes on Variables and Expressions

- **Naming Conventions**: Python uses `snake_case` for variables. Sticking to this rigidly helps with readability.
- **State Management**: Understanding that a variable's value can change over time (mutation) is powerful but requires careful tracking to avoid bugs later on.
- **Order of Operations**: PEMDAS applies in Python. Explicit parentheses are often better than relying on implicit precedence rules to make the code's intent clear to the next reader.
