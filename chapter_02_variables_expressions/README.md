# Chapter 2: Variables, Expressions, and Statements

This chapter moves beyond static text to dynamic state. Variables are the foundation of storing and manipulating data.

## Core Concepts
- **Values and Types**: Strings, integers, floats.
- **Variables**: Named references to values in memory.
- **Statements vs. Expressions**: An expression evaluates to a value (e.g., `1 + 1`). A statement executes an instruction (e.g., `x = 2`).

## Engineering Takeaways
- Naming is critical. `time_elapsed_seconds` is infinitely better than `t`.
- Avoid magic numbers. Use variables to give context to values.
