# Notes on Interface Design

- **The Importance of Refactoring**: Refactoring isn't a sign of failure; it's a natural part of the development lifecycle. Code naturally rots as requirements change. Refactoring is how we fight back.
- **Preconditions and Postconditions**: A clean interface specifies what must be true before calling the function (preconditions) and what the function guarantees upon completion (postconditions).
- **Turtle Graphics**: While simple, Turtle is a fantastic tool for visualizing the flow of execution and the power of parameters.
